Tijdens onze tweede les van elektronica-ict bouwden we verder op de arduino uno.

In de eerste les hebben we deze gesoldeerd.
Nu leerden we hoe we alle gesoldeerde componenten kunnen programmeren en hoe we ze nuttig kunnen gebruiken.
We programmeerde de temperatuurssensor, bewegingssensor en lichtsensor.
Hieronder een code die gebruikt werd om een LED lampje te laten flikkeren. 
void setup(){
	pinMode(10, OUTPUT);
}
void loop(){
	digitalWrite(10, HIGH);
	delay(1000);
	digitalWrite(10, LOW);
	delay(1000);
}


Zelf mochten we hierna nog wat verder coderen en de Arduino Uno mee naar huis nemen.
Het elektronica gedeelte zit er nu op.
