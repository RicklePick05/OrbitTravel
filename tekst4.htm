Tijdens de vierde en laatste les van elektronica-ICT leerden we hoe we met een PHP file moeten werken en hoe dit kunnen combineerbaar is met een HTML file.
We leerden de basis en enkele mogelijkheden zoals checkboxes en tekstvakken maken in een PHP file die we later in onze HTML code zullen zetten.
We maakten kennis met variabelen en hoe men deze moet invoegen in de code. Alsook met strings, voorwaarden, lussen en arryas. Die ons al grotendeels bekend waren van bij het OPO computationeel denken.

Na de introductie hiervan waren we klaar om een Formulier op te stellen. Aangezien dit een zeer belangrijk aspect is waarvoor een PHP is gemaakt, het verwerken van formulieren.
We leerden er de verschillende elementen in plaatsen, de opmaak van een tabel en hoe de formulieren worden verwerkt en later naar een database zullen worden overgebracht.
Het handige hieraan is dat als je een goed formulier maakt en aan een database koppelt je gemakkelijk door bepaalde voorwaarden in te stellen selectief kan gaan selecteren op bepaalde inputs van gebruikers.

Ten slotte kregen we nog kort info over belangrijke elementen waar we moeten op letten indien we een site maken.
Eenmaal de uitleg afgerond was gingen we zelf aan de slag met het programmeren.
